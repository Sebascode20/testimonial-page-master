Solución al desafío [Testimonial Page](https://devchallenges.io/challenge/testimonial-page) de [devChallenges.io](https://devchallenges.io/).

## Demo

- **En vivo:** https://testimonial-page-master20.vercel.app/

## Descripción

Se trata de una página web responsiva que imita el diseño entregado por el desafío. El objetivo principal era practicar maquetación con HTML semántico y CSS moderno, incluyendo flexbox y grid, asegurando que el sitio se vea bien en móviles, tablets y escritorio.

### Aprendizajes

- Refuerzo de conceptos de **HTML5 semántico** y estructura de documentos.
- Uso de **CSS custom properties** para mantener estilos consistentes.
- Implementación de **Flexbox** y **CSS Grid** para colocar elementos de forma responsiva.
- Trabajo con tipografías web y optimización de imágenes.
- Ajustes de diseño móvil usando media queries y unidades relativas.

### Tecnologías

- HTML5 SEMÁNTICO
- CSS3 (propiedades personalizadas, media queries, flexbox, grid)

## Características

- Diseño fiel al entregable del desafío en diferentes resoluciones.
- Navegación simple y accesible.
- Tipografías y colores correspondentes con el diseño.
- Optimización básica de imágenes incluido en `resources/`.
